# Example Document

This is an example Markdown document to demonstrate md2word conversion.

## Features

### Text Formatting

This is **bold text** and this is *italic text*. You can also use ***bold italic***.

### Lists

Unordered list:
- Item 1
- Item 2
- Item 3

Ordered list:
1. First item
2. Second item
3. Third item

### Code

Inline code: `print("Hello World")`

Code block:

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```
